# 🍔 Food Order Management System

A full-stack Food Order Management System designed to streamline the online food ordering process. The application allows customers to browse menus, place orders, make payments, and track deliveries, while restaurants can efficiently manage kitchen operations and orders.

## 🚀 Features

- User Registration & Login
- Browse Food Menu
- Add Items to Cart
- Place Orders
- Online Payment Integration
- Order Tracking
- Kitchen Order Management
- Delivery Status Updates
- RESTful APIs
- Responsive User Interface
